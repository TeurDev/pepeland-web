# Imágenes del Home

Mete aquí archivos con uno de estos nombres y se mostrarán automáticamente:

- `historia.jpg` → bloque "Sobre el lore"
- `facciones.jpg` → bloque "Sobre las facciones"
- `tutorial.jpg` → bloque "Sobre el tutorial"

Si una imagen no existe, se muestra el placeholder de texto por defecto.

Acepta: `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`

Aspect ratio recomendado: **4:3** (las imágenes se recortan con `object-fit: cover`).
