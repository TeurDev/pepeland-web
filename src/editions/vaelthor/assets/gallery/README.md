# Galería

Mete aquí las capturas/imágenes de la galería.

Luego edita `gallery.js` en la raíz de la edición y añade una línea por foto:

```js
{ archivo: 'nombre.jpg', titulo: 'Título', descripcion: 'Texto...', destacada: true },
```

- `archivo` → el nombre exacto del fichero que dejaste aquí
- `destacada: true` → aparece en el carousel de arriba
- Si el archivo no existe, la entrada se ignora automáticamente

Formatos aceptados: `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`
