# Assets públicos

Aquí se ponen archivos estáticos que se sirven tal cual:

- **`favicon.png`** — icono de pestaña del navegador (16x16, 32x32 o 64x64).
  Referenciado en `index.html` como `/favicon.png`.

- **`og-image.png`** — imagen del preview al compartir el link (1200x630 px).
  Referenciada en `index.html` para Open Graph / Twitter Card.

Todo lo que pongas aquí queda accesible en `/nombre-de-archivo` en producción.
